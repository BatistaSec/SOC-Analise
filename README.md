# SOC Monitor Simples - Visual

Ferramenta visual em Python para análise e monitoramento de segurança de redes e aplicações web, simulando operações de um SOC (Security Operations Center) Simples.

## Funcionalidades

- **Scan de Portas:** Descubra portas abertas em um alvo.
- **Exploração Web:** Detecte endpoints sensíveis e respostas suspeitas.
- **Monitoramento de Endpoint:** Verifique disponibilidade de hosts e serviços web.
- **Envio de logs para SIEM:** Integração via syslog UDP.
- **Interface gráfica moderna:** Visualização fácil dos resultados, com ícones e animações.
